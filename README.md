# Pro Max Bot
🔥 Telegram Bot with Welcome, Warn, Remove Warn, Ban/Unban, AI chat, Quiz, Anti-Spam  

## Features
- Welcome new members with rules  
- Warn / Remove warn (command + button)  
- Ban / Unban users  
- Anti-spam detection  
- AI Chat (OpenAI GPT)  
- Quiz with JobQueue  
- Leaderboard  
- Auto delete messages after 2 minutes  

## How to Run
1. Install requirements: `pip install -r requirements.txt`  
2. Set environment variables:  
   - BOT_TOKEN = your Telegram bot token  
   - OPENAI_API_KEY = your OpenAI API key  
3. Run the bot: `python mybot.py`
